# 🎓 AI-Driven Student Performance Prediction System

A Machine Learning project that predicts student performance based on factors such as parental education, test preparation, lunch type, and exam scores.

# Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- Joblib

# Files
- `Ai_student_performance_prediction_project.ipynb`
- `app.py`
- `student_performance_model.pkl`

# Run the Project
```bash
streamlit run app.py
```
